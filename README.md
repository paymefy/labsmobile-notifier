LabsMobile Notifier
===============

Provides [LabsMobile](https://www.labsmobile.com) integration for Symfony Notifier.

DSN example
-----------

```
LABSMOBILE_DSN=labsmobile://USERNAME:API_TOKEN@default?from=FROM
```

where:
- `USERNAME` is your LabsMobile account username
- `API_TOKEN` is an API token, or your Labsmobile account password
- `FROM` is your sender
